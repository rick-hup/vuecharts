Radar charts display multivariate data on axes radiating from a center point.

::chart-demo{name="Simple Radar Chart" description="A basic radar chart with polygon grid." src="radar-charts/simple-radar-chart"}
::

## Compare multiple series

Add multiple `<Radar>` components:

```vue
<Radar data-key="A" stroke="#8884d8" fill="#8884d8" :fill-opacity="0.6" />
<Radar data-key="B" stroke="#82ca9d" fill="#82ca9d" :fill-opacity="0.6" />
```

## Use circular grid

Set `grid-type="circle"` on `<PolarGrid>`:

```vue
<PolarGrid grid-type="circle" />
```

## Show dots on vertices

```vue
<Radar data-key="A" stroke="#8884d8" fill="#8884d8" :fill-opacity="0.6" :dot="{ r: 4, fillOpacity: 1 }" />
```

## Radar props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `dataKey` | `string \| number \| Function` | **required** | Key from data |
| `fill` | `string` | `'#808080'` | Fill color |
| `stroke` | `string` | — | Stroke color |
| `fillOpacity` | `number` | `0.6` | Fill opacity |
| `dot` | `boolean \| object` | `false` | Show dots on vertices |
| `hide` | `boolean` | `false` | Hide the radar |
| `connectNulls` | `boolean` | `false` | Connect across null values |
| `isAnimationActive` | `boolean` | `true` | Enable animation |
