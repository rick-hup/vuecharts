# Introduction

**vccs** (Vue Charts Components) is an unofficial Vue 3 port of [Recharts](https://recharts.org/). It provides composable charting components built with Vue 3 Composition API + JSX/TSX.

## Features

- **Composable** — Build charts using declarative components
- **Vue 3 Native** — Leverages Composition API and reactivity system
- **Recharts Compatible** — Consistent with the Recharts API
- **TypeScript** — Full type support
- **Animated** — Built-in smooth transitions with motion-v
- **Themeable** — Customizable via CSS variables

## Quick Example

```vue
<script setup>
import { AreaChart, Area, XAxis, YAxis, CartesianGrid, Tooltip } from 'vccs'

const data = [
  { name: 'Jan', uv: 4000 },
  { name: 'Feb', uv: 3000 },
  { name: 'Mar', uv: 2000 },
  { name: 'Apr', uv: 2780 },
  { name: 'May', uv: 1890 },
  { name: 'Jun', uv: 2390 },
]
</script>

<template>
  <AreaChart :width="500" :height="300" :data="data">
    <CartesianGrid stroke-dasharray="3 3" />
    <XAxis data-key="name" />
    <YAxis />
    <Tooltip />
    <Area type="monotone" data-key="uv" stroke="#8884d8" fill="#8884d8" />
  </AreaChart>
</template>
```
