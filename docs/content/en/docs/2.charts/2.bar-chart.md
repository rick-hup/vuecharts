# Bar Chart

Bar charts use rectangular bars to show comparisons across categories. The length of each bar is proportional to the data value it represents.

## Usage

```vue
<script setup>
import { Bar, BarChart, CartesianGrid, Tooltip, XAxis, YAxis } from 'vccs'

const data = [
  { name: 'Jan', value: 400 },
  { name: 'Feb', value: 300 },
  { name: 'Mar', value: 600 },
  { name: 'Apr', value: 800 },
  { name: 'May', value: 500 },
  { name: 'Jun', value: 700 },
]
</script>

<template>
  <BarChart :width="500" :height="300" :data="data">
    <CartesianGrid stroke-dasharray="3 3" />
    <XAxis data-key="name" />
    <YAxis />
    <Tooltip />
    <Bar data-key="value" fill="#8884d8" radius="4" />
  </BarChart>
</template>
```

## Props

### BarChart

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `data` | `Array` | `[]` | The data array for the chart |
| `width` | `number` | - | Width of the chart |
| `height` | `number` | - | Height of the chart |
| `margin` | `object` | `{ top: 5, right: 5, bottom: 5, left: 5 }` | Margins around the chart |
| `layout` | `'horizontal' \| 'vertical'` | `'horizontal'` | Layout direction |
| `barGap` | `number \| string` | `4` | Gap between bars in a category |
| `barCategoryGap` | `number \| string` | `'10%'` | Gap between bar categories |
| `syncId` | `string` | - | Sync tooltip/hover across charts |

### Bar

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `dataKey` | `string \| number \| Function` | **required** | Key from data to plot |
| `fill` | `string` | `'#3182bd'` | Bar fill color |
| `radius` | `number \| [number, number, number, number]` | `0` | Border radius of bars |
| `stackId` | `string` | - | Stack ID for stacking bars |
| `maxBarSize` | `number` | - | Maximum bar width |
| `minPointSize` | `number` | `0` | Minimum bar height |
| `hide` | `boolean` | `false` | Hide the bar |
| `isAnimationActive` | `boolean` | `true` | Enable animation |
