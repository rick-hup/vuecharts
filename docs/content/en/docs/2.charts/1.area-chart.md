# Area Chart

Area charts are used to represent the magnitude of change over time. They are similar to line charts, but the area below the line is filled with color.

## Usage

```vue
<script setup>
import { Area, AreaChart, CartesianGrid, Tooltip, XAxis, YAxis } from 'vccs'

const data = [
  { name: 'Jan', value: 400 },
  { name: 'Feb', value: 300 },
  { name: 'Mar', value: 600 },
  { name: 'Apr', value: 800 },
  { name: 'May', value: 500 },
  { name: 'Jun', value: 700 },
]
</script>

<template>
  <AreaChart :width="500" :height="300" :data="data">
    <CartesianGrid stroke-dasharray="3 3" />
    <XAxis data-key="name" />
    <YAxis />
    <Tooltip />
    <Area
      type="monotone"
      data-key="value"
      stroke="#8884d8"
      fill="#8884d8"
      fill-opacity="0.3"
    />
  </AreaChart>
</template>
```

## Props

### AreaChart

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `data` | `Array` | `[]` | The data array for the chart |
| `width` | `number` | - | Width of the chart (use `ResponsiveContainer` for responsive) |
| `height` | `number` | - | Height of the chart |
| `margin` | `object` | `{ top: 5, right: 5, bottom: 5, left: 5 }` | Margins around the chart |
| `layout` | `'horizontal' \| 'vertical'` | `'horizontal'` | Layout direction |
| `syncId` | `string` | - | Sync tooltip/hover across charts with same ID |

### Area

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `dataKey` | `string \| number \| Function` | **required** | Key from data to plot |
| `type` | `CurveType` | `'linear'` | Interpolation type (`monotone`, `basis`, etc.) |
| `fill` | `string` | `'#3182bd'` | Fill color |
| `stroke` | `string` | `'#3182bd'` | Stroke color |
| `fillOpacity` | `number` | `0.6` | Fill opacity |
| `stackId` | `string` | - | Stack ID for stacking areas |
| `hide` | `boolean` | `false` | Hide the area |
| `isAnimationActive` | `boolean` | `true` | Enable animation |
