# 介绍

**vccs**（Vue Charts Components）是 [Recharts](https://recharts.org/) 的非官方 Vue 3 移植版本。它提供了一套可组合的图表组件，基于 Vue 3 Composition API + JSX/TSX 构建。

## 特性

- **可组合** — 使用声明式组件构建图表
- **基于 Vue 3** — 利用 Composition API 和响应式系统
- **Recharts 兼容** — 与 Recharts API 保持一致
- **TypeScript** — 完整的类型支持
- **动画** — 内置流畅的 motion-v 过渡动画
- **主题化** — 可通过 CSS 变量自定义样式

## 快速示例

```vue
<script setup>
import { AreaChart, Area, XAxis, YAxis, CartesianGrid, Tooltip } from 'vccs'

const data = [
  { name: '一月', uv: 4000 },
  { name: '二月', uv: 3000 },
  { name: '三月', uv: 2000 },
  { name: '四月', uv: 2780 },
  { name: '五月', uv: 1890 },
  { name: '六月', uv: 2390 },
]
</script>

<template>
  <AreaChart :width="500" :height="300" :data="data">
    <CartesianGrid stroke-dasharray="3 3" />
    <XAxis data-key="name" />
    <YAxis />
    <Tooltip />
    <Area type="monotone" data-key="uv" stroke="#8884d8" fill="#8884d8" />
  </AreaChart>
</template>
```
