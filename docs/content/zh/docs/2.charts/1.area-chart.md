# 面积图

面积图用于表示随时间变化的幅度。它们类似于折线图，但线下方的区域填充了颜色。

## 用法

```vue
<script setup>
import { Area, AreaChart, CartesianGrid, Tooltip, XAxis, YAxis } from 'vccs'

const data = [
  { name: '一月', value: 400 },
  { name: '二月', value: 300 },
  { name: '三月', value: 600 },
  { name: '四月', value: 800 },
  { name: '五月', value: 500 },
  { name: '六月', value: 700 },
]
</script>

<template>
  <AreaChart :width="500" :height="300" :data="data">
    <CartesianGrid stroke-dasharray="3 3" />
    <XAxis data-key="name" />
    <YAxis />
    <Tooltip />
    <Area
      type="monotone"
      data-key="value"
      stroke="#8884d8"
      fill="#8884d8"
      fill-opacity="0.3"
    />
  </AreaChart>
</template>
```

## Props

### AreaChart

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `data` | `Array` | `[]` | 图表数据数组 |
| `width` | `number` | - | 图表宽度（使用 `ResponsiveContainer` 实现响应式） |
| `height` | `number` | - | 图表高度 |
| `margin` | `object` | `{ top: 5, right: 5, bottom: 5, left: 5 }` | 图表周围的边距 |
| `layout` | `'horizontal' \| 'vertical'` | `'horizontal'` | 布局方向 |
| `syncId` | `string` | - | 同步 ID，用于多图表联动 |

### Area

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `dataKey` | `string \| number \| Function` | **必填** | 数据中要绘制的字段名 |
| `type` | `CurveType` | `'linear'` | 插值类型（`monotone`、`basis` 等） |
| `fill` | `string` | `'#3182bd'` | 填充颜色 |
| `stroke` | `string` | `'#3182bd'` | 描边颜色 |
| `fillOpacity` | `number` | `0.6` | 填充透明度 |
| `stackId` | `string` | - | 堆叠 ID |
| `hide` | `boolean` | `false` | 隐藏面积 |
| `isAnimationActive` | `boolean` | `true` | 启用动画 |
