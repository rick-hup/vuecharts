# 柱状图

柱状图使用矩形柱来展示类别之间的比较。每个柱的长度与其代表的数据值成正比。

## 用法

```vue
<script setup>
import { Bar, BarChart, CartesianGrid, Tooltip, XAxis, YAxis } from 'vccs'

const data = [
  { name: '一月', value: 400 },
  { name: '二月', value: 300 },
  { name: '三月', value: 600 },
  { name: '四月', value: 800 },
  { name: '五月', value: 500 },
  { name: '六月', value: 700 },
]
</script>

<template>
  <BarChart :width="500" :height="300" :data="data">
    <CartesianGrid stroke-dasharray="3 3" />
    <XAxis data-key="name" />
    <YAxis />
    <Tooltip />
    <Bar data-key="value" fill="#8884d8" radius="4" />
  </BarChart>
</template>
```

## Props

### BarChart

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `data` | `Array` | `[]` | 图表数据数组 |
| `width` | `number` | - | 图表宽度 |
| `height` | `number` | - | 图表高度 |
| `margin` | `object` | `{ top: 5, right: 5, bottom: 5, left: 5 }` | 图表周围的边距 |
| `layout` | `'horizontal' \| 'vertical'` | `'horizontal'` | 布局方向 |
| `barGap` | `number \| string` | `4` | 同类别中柱之间的间距 |
| `barCategoryGap` | `number \| string` | `'10%'` | 类别之间的间距 |
| `syncId` | `string` | - | 同步 ID，用于多图表联动 |

### Bar

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `dataKey` | `string \| number \| Function` | **必填** | 数据中要绘制的字段名 |
| `fill` | `string` | `'#3182bd'` | 柱的填充颜色 |
| `radius` | `number \| [number, number, number, number]` | `0` | 柱的圆角半径 |
| `stackId` | `string` | - | 堆叠 ID |
| `maxBarSize` | `number` | - | 最大柱宽 |
| `minPointSize` | `number` | `0` | 最小柱高 |
| `hide` | `boolean` | `false` | 隐藏柱 |
| `isAnimationActive` | `boolean` | `true` | 启用动画 |
